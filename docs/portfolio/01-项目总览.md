# 01 项目总览

## 一句话定义

这是一个面向 A 股场景的 AI 财报分析 Agent，支持单公司解读、双公司对比和通用财务问答，并具备上下文记忆、失败兜底和可视化输出能力。

## 目标用户

- 财经信息初学者：希望用大白话快速理解公司经营质量。  
- 轻度投资关注者：希望快速比较两家公司基本面而非只看股价。  
- AI 产品/技术评估者：希望验证 Agent 在垂直场景的可落地形态。

## 核心价值

1. 把“财报数据 -> 可读解释”流程产品化，降低理解门槛。  
2. 提供对比场景完整输出（双 K 线、对比财务表、文字结论）。  
3. 对异常情况（未识别公司、API失败、数据缺失）提供稳定兜底。  
4. 沉淀评测与日志机制，支持可量化迭代和面试可讲述性。

## 当前阶段成果

- 已实现三路由：单公司 / 双公司对比 / 通用问答。  
- 已实现 Agent 执行流程可视化（步骤追踪、告警、建议问题）。  
- 已建立基线评测资产（测试集 + 自动评测脚本 + 指标体系）。

## 项目边界

### 做什么
- 财报数据解释与对比分析。  
- 可视化辅助理解（K线、评分、对比表）。  
- 合规提示与风险免责声明。

### 不做什么
- 不提供买卖建议。  
- 不承诺收益。  
- 不进行股价预测或交易信号生成。
