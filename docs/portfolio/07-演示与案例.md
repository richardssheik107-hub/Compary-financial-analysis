# 07 演示与案例

## 演示目标

希望面试官在 2 分钟内看到以下核心能力：
1. 系统能理解不同问题意图并自动分流。  
2. 双公司对比不是“文字拼接”，而是有图表与财务证据。  
3. 失败场景下系统仍有可用输出。

## 标准演示 Case

### Case 1：单公司
- 输入：`贵州茅台现在经营质量怎么样，未来一年看什么`  
- 预期亮点：白话解读、评分、雷达图、K线、核心财务表、执行流程可见。

### Case 2：双公司对比
- 输入：`600519和300750谁更好`  
- 预期亮点：领先结论、双K线（上下布局）、财务对比表、对比点评与未来看法。

### Case 3：通用问答
- 输入：`利润增长但现金流下降说明什么`  
- 预期亮点：不指定公司也能稳定输出方法型结论与后续追问建议。
