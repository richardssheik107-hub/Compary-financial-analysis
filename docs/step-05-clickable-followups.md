# Step 5 说明文档：追问建议一键触发

## 1. 目标

将 Step 4 的“建议继续追问”从静态文本升级为可交互按钮，降低用户追问成本。

## 2. 本步改动

- `app.py`
  - 在“建议继续追问”区域增加 `继续问` 按钮
  - 点击后将追问文本写入输入框并自动触发新一轮分析
  - 使用 `st.session_state` 维护：
    - `query_input`：输入框当前值
    - `auto_run_query`：待自动执行问题

## 3. 交互流程

1. 用户先发起一次对比分析。
2. 页面展示追问建议列表，每条建议旁有 `继续问` 按钮。
3. 点击按钮后：
  - 文本自动回填输入框
  - 系统自动执行该追问，无需手动复制粘贴和二次提交

## 4. 验收标准

- 对比场景下可见追问按钮。
- 点击按钮后页面自动刷新并执行对应追问。
- 单公司分析流程不受影响。

## 5. 测试指引

- 先输入：`宁德时代和比亚迪谁更好`
- 在“建议继续追问”中点击任意 `继续问`
- 预期：
  - 输入框内容变为该追问文本
  - 自动触发新一轮分析
