# Step 14 说明文档：顶部上下文栏实时刷新修复

## 1. 问题

- 提交问题后，顶部“当前上下文公司/最近对比对”没有同步更新。

## 2. 原因

- 上下文栏在脚本前段先渲染，提交处理在后段执行。
- 同一轮渲染中，上下文栏读到的是提交前状态。

## 3. 修复方案

- 使用容器占位：
  - 在页面顶部先创建 `context_slot = st.container()`
  - 表单提交与状态更新完成后，再在 `context_slot` 中渲染上下文栏
- 这样保持“显示位置在顶部”，同时读到“本轮最新状态”。

## 4. 验收标准

- 提交公司分析后，上下文栏立即显示最新公司。
- 提交对比问题后，上下文栏立即显示最新对比对。
- 不依赖额外手动刷新。
