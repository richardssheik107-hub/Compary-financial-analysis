# 大白话财报 MVP 需求说明

## 产品定位

大白话财报是一个面向普通大众投资者的财报理解工具，帮助用户用自然语言查询 A 股公司，并获得通俗、谨慎、可视化的财务解读。

## MVP 目标

- 在 Windows 本地通过一键脚本运行。
- 支持用户输入公司名或自然语言问题。
- 优先识别 A 股国民级企业，例如贵州茅台、宁德时代、中国平安、招商银行、比亚迪。
- 优先使用 AKShare 获取公开财务数据。
- 数据接口失败时使用内置样例兜底。
- 使用真实大模型 API 生成白话财报解读。
- 展示企业健康度雷达图和 AI 情绪指示器。
- 明确不提供证券买卖建议。

## 非 MVP 范围

- 不做用户登录。
- 不做交易下单。
- 不做自选股组合管理。
- 不做历史回测。
- 不做实时行情推送。
- 不打包成独立 EXE。

## 成功标准

- 双击 `start.bat` 后应用可以在本机浏览器打开。
- 输入“帮我看看贵州茅台最近赚不赚钱，未来怎么样？”可以获得完整结果。
- 缺少 API Key、AKShare 失败、网络异常时应用不崩溃。
- 页面始终显示风险提示。
- 输出避免直接交易指令。
