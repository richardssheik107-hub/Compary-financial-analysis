# Step 3 说明文档：双公司对比 Agent 流程

## 1. 目标

在现有单公司分析流程基础上，新增“对比类问题”专用执行路径。  
当用户输入“对比/比较/谁更好”类问题时，Agent 自动识别两个公司并输出对比结论。

## 2. 本步改动

- `src/agent.py`
  - 新增对比目标提取 `_extract_compare_targets`
  - 新增对比汇总 `_build_compare_summary`
  - 在 `run_financial_agent` 中新增“财报对比”分支
- `src/models.py`
  - `AgentResult` 新增 `comparison` 字段
- `app.py`
  - 新增 `render_compare_result`，展示双公司对比卡片和对比表格

## 3. 执行流程（对比场景）

1. 意图识别为“财报对比”
2. 提取候选目标并识别两个有效 A 股公司
3. 分别调用数据抓取和 AI 解读
4. 聚合生成对比结论（领先方 + 原因 + 指标表）
5. 页面展示对比结果，同时保留原有单公司渲染兼容

## 4. 验收标准

- 输入如“宁德时代和比亚迪谁更好”可输出：
  - 对比目标识别步骤
  - 工具调用明细
  - 双公司对比结论卡片
  - 指标对比表格
- 单公司问题不受影响，仍走原流程。
- 识别不足两个公司时给出明确提示，不崩溃。

## 5. 测试指引

- 对比问题：`宁德时代和比亚迪谁更好`
- 单公司回归：`贵州茅台`
- 失败场景：`A和B谁更好`（预期提示无法识别两个有效公司）

## 6. 下一步

- Step 4 可扩展：
  - 对比结论增加行业维度说明
  - 对比模式返回两家公司各自完整解读摘要
  - 对比提取规则支持更复杂口语表达
