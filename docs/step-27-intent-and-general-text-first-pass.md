# Step 27：第一步优化（单公司意图识别 + 通用文本质量）

## 目标

在不破坏现有 UI 和主要功能的前提下，完成第一轮核心质量优化：
1. 提升单公司问题的意图识别稳定性。
2. 提升通用问答模式的文字可读性与完整性。

## 本步改动

1. `src/agent.py`
- 重构意图识别逻辑，明确区分：
  - 单公司（含 6 位代码直达）
  - 双公司对比
  - 通用问答
- 保留并强化上下文继承逻辑，追问时可自动补全目标公司。
- 保留原有 `AgentResult` 结构与 compare 路由，避免前端渲染回归。

2. `src/ai_client.py`
- 重构通用问答与兜底文案为可读中文。
- 增加基础文本清洗（重复标点、句尾补全、短文本补全）。
- 保持原有分析接口不变（`analyze_company` / `analyze_general_question`）。

## 回归检查

1. 编译检查：`python -m compileall src/agent.py src/ai_client.py app.py`
2. 路由检查：单公司 / 对比 / 通用问答三类输入均可返回结果
3. 服务检查：`http://localhost:8502/_stcore/health = ok`

## 风险与说明

- 本步未改 `app.py`，UI 样式与页面结构未触碰。
- 文本质量改进属于第一轮，后续还需基于评测集继续优化通用问答质量。
