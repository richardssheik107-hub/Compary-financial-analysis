# Step 7 说明文档：无具体公司问题的通用模式

## 1. 目标

解决“用户未提到具体公司时无法输出结果”的体验问题。  
新增通用问题模式：即使无法识别公司，也能给出可执行的财报分析建议。

## 2. 本步改动

- `src/ai_client.py`
  - 新增 `analyze_general_question(user_question)`：
    - 有 API Key 时走真实模型
    - 无 API Key 时走本地兜底
    - 输出结构与公司解读模式一致（`AnalysisResult`）
- `src/agent.py`
  - 在单公司路径中，若公司识别失败：
    - 不直接返回失败
    - 自动调用 `analyze_general_question`
    - 构造 `general_snapshot` 并返回可渲染结果
- `app.py`
  - `SOURCE_LABELS` 增加 `general_guidance -> 通用问题模式`

## 3. 交互效果

- 用户输入泛化问题（如“现在整体市场风险怎么看”）时：
  - 页面展示 Agent 步骤：意图识别 -> 公司识别与数据抓取 -> 通用问题解读
  - 结果区不报错，给出白话建议、风险提示和下一步提问方向

## 4. 验收标准

- 通用问题能返回结构化结果，不再停在“未识别公司”错误态。
- 单公司和双公司流程继续正常。
- 保持“不构成投资建议”的风险提示。
