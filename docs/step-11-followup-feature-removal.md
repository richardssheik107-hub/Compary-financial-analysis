# Step 11 说明文档：移除追问按钮与自动追问链路

## 1. 目标

按当前产品策略做减法，移除“追问问题”交互能力，保留核心分析链路和建议文本展示。

## 2. 本步改动

- `app.py`
  - 删除“继续问/追问”按钮
  - 删除 `_queue_followup` 回调
  - 删除 `pending_followup` / `auto_run_query` 会话状态及自动触发分支
  - 保留“建议继续追问/你可以继续这样问”的纯文本建议展示

## 3. 当前行为

- 用户可继续手动输入新问题
- 页面不再自动回填问题或自动触发新一轮分析
- 上下文记忆能力仍保留（用于理解连续手动提问）

## 4. 验收标准

- 页面中不再出现追问按钮
- 手动提交分析流程正常
- 编译通过：`python -m compileall app.py`
