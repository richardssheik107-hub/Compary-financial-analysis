# Step 22：双公司对比展示升级

## 目标

把“可识别双公司对比”升级为“可读、可看、可解释”的对比结果页。

## 本步改动

1. 对比场景专用渲染
- 文件：`app.py`
- 新增 `_render_compare_view(comparison)`，在 `agent_result.comparison` 存在时走对比分支渲染。

2. 双K线图展示
- 对比页中并排展示两家公司的近90天K线图（含 MA5/MA10/MA20）。
- 若对比对象缺失行情数据，页面给出对应提示，不中断整体输出。

3. 对比财务报表
- 使用 `comparison.rows` 生成对比表格。
- 表头动态替换为两家公司名称，提升可读性。

4. 对比文字点评
- 输出“领先公司 + 原因 + 可信度 + 限制项”的一段完整点评。
- 让文字部分从“单公司解读”切换为真正的“公司A vs 公司B”表达。

## 验收标准

1. 输入双公司对比问题时，页面出现两张K线图而不是单公司图。
2. 页面出现双公司财务指标对比表格。
3. 点评文字包含两家公司并明确对比结论。
