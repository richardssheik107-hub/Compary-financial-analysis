# 架构一页纸

## 1. 产品目标

构建一个面向 A 股场景的 AI 财报分析 Agent，支持：
- 单公司解读
- 双公司对比
- 通用财务问题问答

## 2. 端到端流程

用户输入  
-> 意图识别与路由（`src/agent.py`）  
-> 工具调用（`build_company_snapshot` / `analyze_company` / `analyze_general_question`）  
-> 结构化结果对象（`src/models.py`）  
-> 前端渲染（`app.py`）  
-> 上下文状态更新（`src/context_store.py`）

## 3. 核心模块

1. Agent 路由层（`src/agent.py`）
- 识别意图（单公司 / 对比 / 通用）
- 编排工具调用顺序
- 输出结构化 `AgentResult`（含步骤追踪、告警、建议问题）

2. 数据层（`src/data_service.py`）
- 公司名称/代码识别
- 财务指标与日线数据获取
- 数据不可用时兜底处理

3. 分析层（`src/ai_client.py`）
- 将结构化财务数据转为大白话文本
- API 异常时本地兜底
- 风险提示与内容安全约束

4. UI 层（`app.py`）
- 单公司视图：评分、雷达图、K线、财务表、文字解读
- 对比视图：双 K 线、对比报表、对比结论与未来看法
- 展示 Agent 执行流程与延申问题建议

5. 上下文层（`src/context_store.py`）
- 维护当前公司、最近对比对、最近意图
- 支持追问时上下文继承

## 4. 稳定性设计

- API / 数据失败的兜底路径
- 单公司 / 对比 / 通用路由清晰分离
- 执行 trace 可观测，便于调试
- 评测基线（`eval/testset.csv` + `scripts/eval_runner.py`）支持持续迭代
