# Step 17 说明文档：会话状态自愈修复

## 1. 问题

版本迭代后，旧会话缓存中的 `last_agent_result` 可能与新结构不兼容，导致页面重跑时崩溃或无法输出内容。

## 2. 修复方案

- 在 `app.py` 新增兼容性校验 `_is_compatible_agent_result(...)`
- 启动时若检测到旧缓存结构不兼容，自动清理 `last_agent_result`
- 渲染阶段增加保护：
  - 若渲染异常，自动清空缓存并提示用户重新提交

## 3. 预期效果

- 避免“旧缓存卡死当前页面”
- 保证服务升级后页面可自动恢复

## 4. 验收标准

- 服务正常启动
- 页面刷新后可恢复可用状态
- 健康检查 `/_stcore/health` 返回 `ok`
