# 本地研究资料目录规范（Step 45）

用于放置合法来源的公司研究资料，支持 `txt`、`md`、`pdf`。

## 推荐命名

`股票代码_公司名_资料类型_日期.扩展名`

示例：
- `600519_贵州茅台_年报摘要_2025Q4.md`
- `300750_宁德时代_研报观点_2026-03.txt`
- `000001_平安银行_季报摘录_2026Q1.pdf`

## 最小内容建议（txt/md）

至少包含以下任意两项：
- 主营业务
- 管理层观点
- 风险提示
- 增长驱动

## 命中机制说明

系统会同时在“文件名 + 文本内容”中匹配：
- 公司名
- 股票代码

若 `pdf_empty > 0`，通常表示 PDF 为扫描版或不可提取文本，建议补一份同名 `txt/md` 摘要。
